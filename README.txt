Tawqeet, Prayer times app.

BASIC EXPLAINATION
This application was made with the purpose to display to the user today's prayer times in Kuala Lumpur Malaysia.
This project contains firebase authentication, firebase storage and http api requests.
It works by first making the user login or signup then the prayer times are fetched from the API then they are displayed on the screen.

TO RUN -> Clone project on your local machine using the link to the repo -> Run "dart pub get" -> ENJOY!
